## 配置相关

* 编码
```_vimrc
set encoding=utf-8
set langmenu=zh_CN.UTF-8
language message zh_CN.UTF-8
set termencoding=utf-8
set fileencodings=utf-8

source $VIMRUNTIME/delmenu.vim
source $VIMRUNTIME/menu.vim
```
